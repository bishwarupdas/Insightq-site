üé® KEY DESIGN PRINCIPLES (gentlerain.ai-inspired)
1. Whitespace is Your Friend
* 120px vertical padding between sections (not 60px)
* Let content breathe
* Don't fill every pixel
2. Typography Does the Heavy Lifting
* Large headlines (64-96px)
* Generous line-height (1.5-1.8)
* Limited font sizes (4-5 total)
* Hierarchy through size, not decoration
3. Minimal Color Usage
* Orange ONLY for CTAs and key highlights
* Navy for dark sections
* White/light gray for most backgrounds
* Don't rainbow everything
4. Show, Don't Abstract
* Real product screenshots > abstract concepts
* Actual use cases > generic benefits
* Specific numbers > vague claims
5. Conversational Copy
* Write like you talk
* No corporate jargon
* Short sentences
* Clear value statements
6. Strategic Animation
* Parallax in 3 sections max (hero, solution, footer)
* Subtle fade-ins on scroll
* Hover states that feel good
* No animation vomit
7. Mobile-First Thinking
* Design for mobile, enhance for desktop
* Touch-friendly targets (48px minimum)
* Readable text without zoom (16px+)
* Simplified interactions
8. Fast Loading
* Optimize all images
* Lazy load below fold
* Minimal dependencies
* Target <2s LCP
